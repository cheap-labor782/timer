# Stopwatch

這是一個使用 Python 的 Tkinter 模組開發的輕量級桌面碼表應用程式。  
具備極簡暗黑風格介面，並支援鍵盤快速鍵操作，適合需要精準計時與紀錄的場景。

## 功能特色

- **暗黑主題 (Dark Mode)**：簡潔的黑底白字設計，保護眼睛且視覺清爽。  
- **鍵盤快捷控制**：無需滑鼠即可操作，提升效率。  
- **Lap 紀錄功能**：支援計時中即時紀錄圈數，並顯示在紀錄清單。  
- **低資源佔用**：輕量化設計，執行順暢。  

## 快速上手

### 鍵盤指令表

| 按鍵 | 功能           |
|------|----------------|
| S    | 開始計時 (Start) |
| P    | 暫停計時 (Pause) |
| R    | 重置計時 (Reset) |
| M    | 紀錄時間 (Record) |

## 執行需求

- Python 3.x  
- 內建 Tkinter 模組（無需額外安裝 pip 套件）

## 如何執行

1. 確保已安裝 Python 3。  
2. 下載所有檔案至同一資料夾。  
3. 開啟終端機 (Terminal/CMD)，進入該資料夾。
